# Hubcell

Hubcell is a powerful container and cell management node. It provides an efficient and secure way to run your workloads.

## Installation

Run the following command to install Hubcell on any supported Linux machine:

```bash
curl -sL https://raw.githubusercontent.com/hubfly-space/hubcell-docs/master/install.sh | sudo bash
```

## Quick Start

Once installed, start the Hubcell node with the web dashboard enabled:

```bash
hubcell serve --web
```

Access the dashboard at `http://localhost:3000` (or the configured port).
