Hubcell Documentation

Welcome to the official documentation for Hubcell, a robust node runner and container management system.

Installation

You can easily install Hubcell via our installation script:

curl -sL https://raw.githubusercontent.com/hubfly-space/hubcell-docs/master/install.sh | sudo bash

Getting Started

Once installed, start the Hubcell node with the web interface: 

hubcell serve --web
